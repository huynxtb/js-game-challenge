Astro Salvage - Text-Based Console Game

Introduction:
You are the captain of a salvage spaceship operating in the aftermath of a destroyed asteroid field. Your mission is to collect valuable salvage debris worth at least 100 units within 20 turns, managing your ship's limited fuel, integrity, and the crew's morale. Each turn, you make critical decisions to move, scan, salvage, repair, or rest while facing random hazards such as asteroid storms, pirates, and equipment failures.

Setup Instructions:
- Ensure you have Node.js installed on your system. You can download it from https://nodejs.org/
- Save the provided JavaScript code into a file named 'astro_salvage.js'.

How to Run:
- Open your command line interface (Terminal, Command Prompt, etc.).
- Navigate to the directory where the file 'astro_salvage.js' is located.
- Run the game by entering:

  node astro_salvage.js

Gameplay:
- Follow on-screen instructions.
- Each turn, choose an action by typing one of: move, scan, salvage, repair, rest
- Monitor your resources and respond to random events.
- The game ends when you reach the target salvage worth, run out of fuel or morale, lose ship integrity, or complete all turns.

Enjoy your strategic space salvage adventure, Captain!
