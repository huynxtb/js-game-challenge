Chrono Custodians - Text-Based Time-Traveling Game

Introduction:
You play as a Chrono Custodian, a time-traveling guardian tasked with preserving the timeline from dangerous anomalies caused by rogue time travelers. Your mission is to identify and neutralize these anomalies before they cause irreversible damage to history.

Gameplay Overview:
- The game is turn-based, with each cycle representing a unit of time in which you take actions.
- Manage your Temporal Energy, which is required to investigate anomalies, travel to their time periods, and contain them.
- Anomalies randomly appear and you must prioritize which to address first.
- The timeline integrity reflects the health of the timeline. It depletes if anomalies persist.

Win Condition:
- Successfully contain all anomalies within 20 time cycles without letting the timeline collapse or energy run out.

Loss Conditions:
- Timeline integrity reaches zero.
- You run out of Temporal Energy while anomalies remain.
- Too many anomalies accumulate unchecked.

Setup and Running:
1. Save the provided code into a file named `chrono_custodians.js`.
2. Ensure you have Node.js installed on your system (https://nodejs.org/).
3. Open a terminal or command prompt.
4. Run the game by executing:
   node chrono_custodians.js

Controls and Interaction:
- Follow on-screen prompts to choose actions each cycle.
- Use numeric input to select options.
- Manage your Temporal Energy carefully to ensure success.

Enjoy preserving the timeline and good luck, Guardian!
