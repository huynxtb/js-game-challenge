ChronoChef Challenge - Text-Based Time Management Game in Node.js Console

Introduction:
ChronoChef Challenge is a whimsical, text-based kitchen game where you act as a master chef racing against the clock to prepare and serve a day's worth of diverse dishes. Manage your time, energy, ingredients, and kitchen tools strategically to complete all orders before the kitchen closes.

Game Features:
- Turn-based play; each turn equals 5 minutes of kitchen time.
- Manage ingredients, cooking tools, and energy resources.
- Handle an order queue with different dishes to prepare.
- Choose tasks such as chopping, cooking, plating, and serving.
- Random events can impact ingredient availability and task durations.
- Win by completing all orders before closing time without running out of energy or essential ingredients.

Setup & Running Instructions:

Requirements:
- Node.js installed (version 12 or higher recommended).

To run the game:
1. Save the code in a file named chrono_chef_challenge.js
2. Open a terminal/command prompt.
3. Navigate to the directory containing chrono_chef_challenge.js.
4. Run the game by typing:

   node chrono_chef_challenge.js

Gameplay Instructions:
- Follow the console prompts.
- Each turn, you'll see your current time, energy, resources, and pending orders.
- You will be asked to choose a task from the current order's pending tasks by entering the task number.
- You can also type 'rest' to regain energy or 'skip' to wait and lose some energy.
- Completing tasks consumes energy and time; manage them carefully.
- Random kitchen events may occur each turn, adding unpredictability.
- Complete all orders before the kitchen closes at 12:00.

Enjoy becoming the ultimate ChronoChef and good luck managing your whimsical kitchen!
