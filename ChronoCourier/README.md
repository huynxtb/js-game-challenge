ChronoCourier - Console-Based Time-Travel Delivery Game

Introduction:
ChronoCourier lets you become a daring courier who travels through time to deliver packages in various historical eras. Each package has a deadline and challenges multiply as you navigate prehistoric landscapes, medieval castles, industrial cities, and futuristic worlds. Manage your stamina carefully to keep moving, rest when necessary, and tactically prioritize your cargo to deliver all packages on time.

Setup Instructions:
- Ensure you have Node.js installed on your system (https://nodejs.org/).
- Save the game source code in a file named 'chronocourier.js'.

How to Play:
- Open your terminal or command prompt.
- Navigate to the directory where 'chronocourier.js' is saved.
- Run the game by typing: node chronocourier.js

Gameplay:
- Each turn you can choose one action:
  1. Move closer to a package's destination (costs stamina depending on era).
  2. Rest to regain stamina.
  3. Rearrange cargo order to prioritize deliveries.
  4. View detailed package information.
  5. Quit the game.
- Random events occur based on the era that can affect your stamina or delay progress.
- Deliver all packages before their deadline to win.
- Running out of stamina before finishing deliveries results in a loss.

Enjoy being the ChronoCourier and mastering time itself!
