Galactic Courier - Console Interstellar Delivery Mission

Introduction:
You are an interstellar courier tasked with delivering critical packages between planets in a volatile galaxy. Manage your fuel, time, and ship integrity carefully while facing random space hazards such as pirate attacks, meteor storms, and engine malfunctions. Successfully deliver all packages within their deadlines and safely return to your home base Terra to win.

Setup Instructions:
1. Ensure you have Node.js installed on your system.
2. Save the provided source code into a file named 'galactic_courier.js'.

How to Run the Game:
- Open your terminal or command prompt.
- Navigate to the directory containing 'galactic_courier.js'.
- Run the game with the command:
    node galactic_courier.js

Gameplay Basics:
- Each turn, you can travel from your current planet to a connected planet by choosing from the available options.
- Traveling consumes fuel based on the distance between planets.
- You may also choose to stay on your current planet to rest, which consumes a turn but may repair your ship.
- Packages have delivery deadlines; each turn reduces their remaining time.
- Random space events may affect your ship's fuel and integrity.
- The game ends when you have delivered all packages and return to Terra, or if you run out of fuel, your ship is destroyed, or you miss any deadline.
